# Set-up / 使い方

- [Make sure Node.JS is installed.](https://nodejs.org/en/ "Node.JS download page")
- install dependencies
	`npm install --save`
	`npm install --save express@4.10.2`
- run `node multiServer.js`
- point your browser to `http://localhost:3000`
